# Clap-Switch
Clap-Switch - moje wykonanie przełącznika, reaguyjącego na pojedyńcze klaśnięcie, pstryknięcie, pozwala na sterowanie (załączanie i wyłączanie) urządzenia podpiętego do złącz przekaźnika.
